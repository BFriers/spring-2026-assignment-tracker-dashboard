# spring-2026-assignment-tracker-dashboard

Project Overview
  Designed and built a structured assignment tracking system in Google Sheets to centralize workload management and monitor deadlines efficiently.

Features
  - Automated status categorization (Completed, In Progress, Overdue)
  - Conditional formatting for deadline alerts
  - Dynamic sorting and filtering
  - Organized task tracking by course
  - Clear dashboard-style layout for visibility

Tools Used
  - Google Sheets
  - Microsoft Excel (.xlsx export)
  - Logical formulas
  - Conditional formatting

Skills Demonstrated
  - Spreadsheet modeling
  - Workflow automation
  - Data organization and structuring
  - Logical problem-solving
